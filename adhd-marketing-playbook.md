## 二、账号定位

### 你的账号该长什么样

#### 选项 A: 「ADHD 爸爸的日记」(推荐)⭐

**人设**:
- 真实 ADHD 孩子的爸爸
- 工程师 + 创业者
- 在做一个帮 ADHD 家庭的 App

- 真实身份,信任度高

**我的推荐**: **选 A**,你有最强差异化(男性 + 工程师 + 真用户)

## 三、内容矩阵(3 大主题)

### 主题 1: 创始人故事(20% 内容)⭐⭐⭐⭐⭐

**最容易爆**,因为有共情。

#### 选题示例

```
- 「孩子被诊断为 ADHD 的那天,我们一家三口的反应」
- 「ADHD 妈妈深夜的痛哭,你经历过吗?」
- 「老公说我太敏感,直到他陪了一天娃」
- 「我们家戒不掉的吼叫,直到我做了这件事」
- 「30 岁程序员,决定为 ADHD 家庭做一个 App」
```

#### 写作公式

```
钩子(第 1 句):
  「昨晚我又对孩子吼了。」
  「凌晨 2 点,我抱着孩子哭了。」

故事(中段):
  具体场景 + 细节 + 情绪
  
共情(结尾):
  「如果你也经历过这个,评论区告诉我」
```

#### 频率

每周 **1-2 篇**(够了,不要每天)

---

### 主题 2: ADHD 知识干货(50% 内容)⭐⭐⭐⭐

**容易涨粉 + 收藏**,但需要真功夫。

#### 选题示例

```
科普类:
- 「ADHD 不是教育出来的,是天生的」
- 「3 个国际权威指南推荐的 ADHD 干预方法」
- 「ADHD 孩子真的需要吃药吗?医生不说的真相」

方法类:
- 「让 ADHD 孩子自主写作业的 5 个方法」
- 「代币奖励法实操指南(附下载模板)」
- 「ADHD 孩子早起穿衣的 3 个小技巧」

工具类:
- 「ADHD 家长必备的 5 个工具」(自然带出自己的产品)
- 「我用这个 Excel 模板帮孩子打卡,30 天有效」

误区类:
- 「这 5 个常见误区,正在伤害你的 ADHD 孩子」
- 「ADHD 不是『性格内向』,别再误诊」
```

#### 写作公式

```
标题: 数字 + 痛点 + 解决
  「3 个方法让 ADHD 孩子自主写作业」

封面: 大字 + 重点信息
  「ADHD 孩子写作业」(超大字)
  + 副标题「3 个亲测有效的方法」

正文:
  开头共情(1 段)
  → 提出问题(1 段)
  → 3-5 个具体方法(每个方法 1 张图)
  → 真实案例(1 段)
  → 引导互动(评论区聊你的经验)
```

#### 频率

每周 **2-3 篇**

---

### 主题 3: 产品成长记(30% 内容)⭐⭐⭐

**Build in Public**,自带流量。

#### 选题示例

```
进度类:
- 「为 ADHD 家庭做的小程序,开发日记 Day 30」
- 「今天上线了任务拆分功能,演示给你看」
- 「用户突破 100,感谢每一位 ADHD 妈妈」

征求反馈类:
- 「ADHD 家长们,你最希望我加什么功能?」
- 「我做的这个 UI,你觉得好看吗?」

幕后类:
- 「程序员爸爸如何业余做出 ADHD 小程序的」
- 「为什么我决定 0 元上线,不收钱?」
```

#### 写作公式

```
透明 + 真诚 + 求反馈

例:
「Day 30 进度:
- ✅ 任务拆分上线了
- ✅ AI 助手能跟孩子聊天
- ⏳ 周报功能下周

求反馈:
1. 你们最缺什么功能?
2. 你愿意付多少钱?

评论区聊聊~」
```

#### 频率

每周 **1 篇**

## 五、爆款笔记的 5 个技巧

### 技巧 1: 标题钩子

**有效的标题模式**:

```
共情型: 「ADHD 妈妈,这个时刻你也经历过吗?」
数字型: 「3 个让 ADHD 孩子专注的方法」
反差型: 「我以为 ADHD 是教育问题,直到...」
痛点型: 「ADHD 孩子写作业,我崩溃了」
争议型: 「ADHD 该不该吃药?医生告诉我...」
```

### 技巧 2: 封面设计

**封面 = 你的 80% 流量**

```
✅ 大字 + 重点信息
✅ 真人照(增加信任)
✅ 对比图(before/after)
✅ 浅色背景 + 高对比文字

❌ 不要小字
❌ 不要纯文字(无图)
❌ 不要 AI 假图(掉粉)
```

### 技巧 3: 黄金前 3 秒

**用户刷小红书 = 0.3 秒决定看不看**

```
封面: 第 1 秒(80% 决定)
标题: 第 2 秒(15% 决定)
首句: 第 3 秒(5% 决定)
```

### 技巧 4: 引导互动

**评论 + 点赞 + 收藏 = 算法推荐**

```
结尾必加:
- 「评论区告诉我你的经验」
- 「同款 ADHD 家长扣 1」
- 「你家娃多大?评论区聊聊」
```

### 技巧 5: 关键词埋好

**让搜索能搜到**:

```
笔记里自然提到:
- ADHD
- 多动症
- 注意力缺陷
- 专注力训练
- 儿童行为管理
- ADHD 妈妈
- ADHD 爸爸
- (你产品名)
```

## 六、引流到小程序

### 合规的引流方式

```
✅ 简介里加「小工具」名(不要直接说小程序)
✅ 笔记结尾「我做了一个免费工具,关注我看」
✅ 私信引导(评论后私信发链接)
✅ 评论区互动(不要广告)

❌ 不要笔记里直接挂二维码(违规)
❌ 不要诱导关注
❌ 不要直接发链接(被限流)
```

### 推荐引流路径

```
小红书笔记 → 引导关注
   ↓
关注后看简介 → 「我做了一个 ADHD 助手小程序」
   ↓
小程序搜索 → 找到你的产品
   ↓
注册使用 → 看是否解决问题
   ↓
真实反馈 → 留存
```

## 九、避坑指南

### ❌ 坑 1: 一上来就推销

**症状**: 第 1 篇笔记就发产品

**正确**: 前 10 篇全是干货 + 故事,第 11 篇才隐约提产品

### ❌ 坑 2: 标题党

**症状**: 标题骇人,内容空洞

**正确**: 标题吸引,内容有干货

### ❌ 坑 3: 用 AI 生成内容

**症状**: 文字 AI 感重,真实感低

**正确**: AI 起草 + 你深度改 + 加真实经历

### ❌ 坑 4: 频率忽高忽低

**症状**: 一天 5 篇,然后 2 周不发

**正确**: 每周 2-3 篇,持续

### ❌ 坑 5: 不互动

**症状**: 发完就走

**正确**: 评论 2 小时内必回,主动私信关键用户

### ❌ 坑 6: 抱怨 / 焦虑 / 负能量

**症状**: 老说「养 ADHD 娃太难了」

**正确**: 有故事 + 有方法 + 有希望

### ❌ 坑 7: 蹭医疗 / 教培热点

**症状**: 借势某教育新闻

**正确**: 保持中立,聚焦 ADHD 干预方法
