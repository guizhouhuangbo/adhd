
## 十、常见上线坑

| 坑 | 对策 |
|---|---|
| ICP 备案被卡(2 个月还没下来) | 提前 25 天启动,网站名朴素 |
| 微信小程序审核被拒 | 避开医疗词汇,准备好审核账号 |
| 上线后接口挂了 | Hertzbeat 告警 + 立即回滚 |
| 用户 0 反馈 | 主动加用户微信问 |
| 自己用着不顺 | 先在自家用 2 周再公测 |
| 焦虑「为啥没人用」 | 这是必经阶段,坚持 6 个月 |
| 服务器费用过高 | 用最小配置,慢慢扩 |
