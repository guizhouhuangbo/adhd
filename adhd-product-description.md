## 一、一句话定位

**「给 ADHD 儿童家长的专注力训练 + 行为管理小程序」**

英文版: *AI-powered focus & behavior helper for ADHD children's parents*

---

## 二、解决什么问题

### 用户痛点(真实场景)

**ADHD 家长每天的崩溃时刻**:

| 时刻 | 痛点 |
|---|---|
| 早上 | 喊孩子起床 10 次,拖延到迟到 |
| 上学路上 | 东西又落了,书包乱七八糟 |
| 学校时段 | 老师又投诉「上课走神/打人」 |
| 放学后 | 写作业 30 分钟的内容,3 小时做不完 |
| 晚饭后 | 不愿洗澡,玩具不收 |
| 睡前 | 不肯放手机,睡眠不规律 |

**家长的真实感受**:
- 😢 「孩子是不是讨厌我?」
- 😡 「我每天都在吼,我都不喜欢自己了」
- 😔 「医生说不影响生活,但每天都崩溃」
- 😞 「身边没人理解,丈夫说我太敏感」
- 😰 「这个孩子未来怎么办?」

### 我们怎么解决

```
痛点                       我们的方案
─────────────────────  ──────────────────────
任务太抽象,孩子做不到  →  AI 自动拆分成 5-7 个具体小步骤
没耐心反复提醒         →  小程序自动提醒 + 可视化倒计时
不知道怎么奖励         →  代币奖励法,科学有效
没人理解,无人支持      →  AI 家长助手 24h 在线共情 + 建议
看不到进步,容易放弃    →  数据可视化,每周进步报告
没专业指导             →  ADHD 知识库 + 专家咨询
```

## 五、核心功能(MVP)

### 5 大模块

#### 模块 1: 🎯 行为打卡 + 代币奖励

- 任务模板库(分年龄段、分场景)
- 自动化提醒
- 完成确认 + 拍照
- 星星奖励
- 实物奖励兑换

#### 模块 2: 🧠 专注力训练游戏

- 持续注意力训练
- 工作记忆训练
- 执行功能训练
- 每天 10-15 分钟

#### 模块 3: ⏰ 时间管理工具

- 可视化倒计时(沙漏/进度条)
- 番茄钟(ADHD 改良版,8-12 分钟)
- 任务拆分助手(AI 自动拆)

#### 模块 4: 🤖 AI 家长助手

- 24h 在线共情对话
- 个性化建议
- 紧急情况应对方案

#### 模块 5: 📊 数据洞察

- 每周/月进步报告
- 行为模式分析
- 触发因素识别(AI 找规律)

## 九、不做什么(明确边界)

为了聚焦,**我们明确不做**:

```
❌ 不做医疗诊断(避开监管)
❌ 不替代医生(明确写在协议里)
❌ 不卖药 / 不推药
❌ 不做学科教育(避开 K12 监管)
❌ 不做硬件(成本高、风险大)
❌ 不做社交(避免儿童社交风险)
❌ 不做视频 / 直播(资质要求高)
❌ MVP 阶段不做 AI 视觉识别(过度工程)
❌ MVP 阶段不做自动续费(合规风险)
```
