## 🎯 创业核心信息(一页速览)

### 创始人画像

```
身份: Java 后端工程师 + ADHD 孩子的父亲
优势: 后端技术强 + 真实用户身份 + 已有 TKE/CVM 资源
约束: 个人开发 + 下班 2 小时 + 没 UI 经验
公司: 已注册个人公司 ✅
```

### 项目定位

**「家长侧的 ADHD 儿童行为管理 + AI 辅助训练工具」**

- ❌ 不是医疗工具
- ❌ 不是教育工具
- ✅ 是家长助手

### 核心策略(2026-06-23 更新)

```
✅ 先做免费版,验证产品价值
✅ 暂不接支付,先看用户量
⏸️ 月活到 500+ 且用户主动问付费时,再接支付
```

### 技术栈

```
前端: 微信小程序原生 + Vant Weapp
后端: Spring Boot 3 + MyBatis-Plus + MySQL 8 + Sa-Token + WxJava
AI:   DeepSeek API(直接 HTTP)
部署: 腾讯云 TKE + APM 自动注入
监控: Hertzbeat + 钉钉告警
```

### 12 周路线图

```
Week 1-4:  沉浸用户 + 备案 + 手动版 MVP
Week 5-8:  后端开发(Spring Boot)
Week 9-12: 小程序开发(微信原生 + Vant)
Week 13-17: 内测 + 公测
Week 18+:  持续运营 + 视量定付费时机
```
